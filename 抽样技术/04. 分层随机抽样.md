# 分层随机抽样

## 基本问题

考试最多的计算题在本章

### 什么是分层随机抽样

N = N1 + N2 + ... + NL

#### 特点

1. 提高估计精度
2. 可对各层进行估计
3. 便于组织

#### 分层原则

1. 层内方差尽可能小
2. 层间方差尽可能大

### 符号

```
第i层单位数
单位数总和
第i层均值
第i层方差
第i层权数
第i层抽样比
```

> 等比例抽样
> 不等比例抽样
